**Types:**
```typescript
// Operation type union — each operation is a tagged object
type ResizeOp = { type: 'resize'; width?: number; height?: number; fit?: 'cover' | 'contain' | 'fill' };
type CropOp = { type: 'crop'; x: number; y: number; width: number; height: number };
type AspectCropOp = { type: 'aspectCrop'; ratio: '1:1' | '16:9' | '9:16' | '4:3' | '3:4'; gravity?: 'center' | 'north' | 'south' | 'east' | 'west' };
type CircleMaskOp = { type: 'circleMask' };

export type ProcessingOperation = ResizeOp | CropOp | AspectCropOp | CircleMaskOp;

export interface ImageInfo {
  width: number;
  height: number;
  format: string;
  channels: number;
}

export interface ProcessingResult {
  buffer: Buffer;
  originalInfo: ImageInfo;
  outputInfo: ImageInfo;
  operationsApplied: string[];
}
```

**Functions:**

`getImageInfo(buffer: Buffer): Promise<ImageInfo>` — Uses `sharp(buffer).metadata()` to extract width, height, format, channels.

`resize(buffer: Buffer, op: ResizeOp): Promise<Buffer>` — Uses `sharp(buffer).resize({ width: op.width, height: op.height, fit: op.fit || 'cover' }).png().toBuffer()`. The `fit` parameter maps directly to sharp's fit option (cover = crop to fill, contain = letterbox, fill = stretch).

`crop(buffer: Buffer, op: CropOp): Promise<Buffer>` — Uses `sharp(buffer).extract({ left: op.x, top: op.y, width: op.width, height: op.height }).png().toBuffer()`.

`aspectCrop(buffer: Buffer, op: AspectCropOp): Promise<Buffer>` — Calculates crop dimensions from the source image metadata:
  - Parse ratio string to get target aspect (e.g., '16:9' → 16/9)
  - Get source dimensions via `sharp(buffer).metadata()`
  - Calculate largest rectangle of target aspect that fits within source
  - If source is wider than target ratio: crop width (keep full height, center horizontally)
  - If source is taller than target ratio: crop height (keep full width, center vertically)
  - Apply gravity offset: 'center' = centered (default), 'north' = top-aligned, 'south' = bottom-aligned, 'east' = right-aligned, 'west' = left-aligned
  - Use `sharp(buffer).extract({ left, top, width, height }).png().toBuffer()`

`circleMask(buffer: Buffer): Promise<Buffer>` — Creates circular mask with transparent background:
  - Get source dimensions via `sharp(buffer).metadata()`
  - Use the smaller of width/height as diameter
  - Create SVG circle mask: `<svg><circle cx="${r}" cy="${r}" r="${r}" fill="white"/></svg>` where r = diameter/2
  - First resize/crop source to square (diameter x diameter, centered)
  - Then composite the circle mask using `sharp(squareBuffer).composite([{ input: maskBuffer, blend: 'dest-in' }]).png().toBuffer()`
  - Ensure output has alpha channel: use `.ensureAlpha()` before compositing

`applyOperations(buffer: Buffer, operations: ProcessingOperation[]): Promise<ProcessingResult>` — Pipeline function:
  - Get originalInfo from input buffer
  - Loop through operations array in order, applying each function to the current buffer
  - For each operation, dispatch to the correct function based on `type` field
  - Track operation names in operationsApplied array (e.g., "resize(200x200, cover)", "circleMask")
  - After all operations, get outputInfo from final buffer
  - Return { buffer, originalInfo, outputInfo, operationsApplied }

**Important implementation notes:**
- All output must be PNG (`.png()` before `.toBuffer()`) — transparency needed for circle masks
- Each function is standalone (takes Buffer, returns Buffer) for testability
- `applyOperations` is the composition layer — it calls individual functions sequentially
- Default gravity for aspectCrop is 'center'
- Default fit for resize is 'cover'
